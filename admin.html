Comment ajouter une création — étape par étape

    
    
      
  1. 
        Prépare ton image : Renomme-la simplement (ex: bouquet-roses.jpg), puis place-la dans le dossier images/creations/.
      
    2. 
      
  2. 
        Remplis le formulaire ci-dessus avec le titre, la description, la catégorie, le nom de l'image et la date.
      
    3. 
      
  3. 
        Clique sur "Générer le code" — un bloc de texte apparaît.
      
    4. 
      
  4. 
        Ouvre le fichier creations.json avec un éditeur de texte (Bloc-notes, par exemple).
      
    5. 
      
  5. 
        Ajoute le bloc généré juste après le premier [ du fichier (au tout début de la liste), suivi d'une virgule.
      
    6. 
      
  6. 
        Enregistre le fichier. Rafraîchis ton site — ta nouvelle création apparaît !
